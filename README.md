# Cohabia

Cohabia es una aplicación nativa para Android diseñada para facilitar la convivencia en viviendas compartidas. La app centraliza en un solo lugar la organización de tareas, la gestión de gastos comunes y el control de dispositivos del hogar, permitiendo que todos los habitantes estén sincronizados en tiempo real.

### Funcionalidades principales

* **Gestión de vivienda:** Creación de grupos de convivencia mediante códigos de invitación.
* **Control de electrodomésticos:** Registro de dispositivos y monitorización de programas (inicio, pausa y reinicio) desde el móvil.
* **Gastos compartidos:** Registro de pagos y división automática de deudas entre los miembros de la casa.
* **Lista de la compra:** Gestión de productos por categorías con actualización en tiempo real.
* **Notas comunes:** Sistema de anotaciones compartidas con guardado automático.

### Tecnologías utilizadas

El proyecto utiliza un stack moderno para garantizar fluidez y estabilidad:

* **Lenguaje:** Kotlin
* **Interfaz:** Jetpack Compose (Material 3)
* **Backend:** Firebase (Firestore, Auth, Storage y Cloud Messaging)
* **Inyección de dependencias:** Koin
* **Animaciones:** Lottie
